Browserstack & Nodemon

A JS App running on node.js via Nodemon. A gulp script defines two tasks to self re-compoile the app and refresh browser whenever its components get changed.

See gulpfile.js, app.js, public/index.html and public/site.css

checkout project install dependencies: npm install start the app: gulp change app.js, public/index.html and public/site.css and enjoy the lifecycle.
